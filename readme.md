
# 整理目录中的教程文件
> 1.教程文件如果总体积大于500M则以不超过500M为标准分文文件夹保存  
> 2.条件1中的最子级文件夹以此文件夹保存的教程视频序号区间命名  
> 3.单个系列教程的根目录生成所有教程的文件列表文档  
> 4.若文件名没有序号则添加序号
